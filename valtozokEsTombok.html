Változók és tömbök

Változó típusok

Numerikus adatok

  • Boolean (bool) -> igaz vagy hamis értéket
  • Byte (byte) -> előjel nélküli 8 bit
  • SByte (sbyte) -> előjeles 8 bit
  • Int16 (short) -> előjeles 16 bit
  • UInt16 (ushort) -> előjel nélküli 16 bit
  • Int32(int) -> előjeles 32 bit
  • UInt32 (uint) -> előjel nélküli 32 bit
  • Int64 (long) -> előjeles 64 bit
  • UInt64 (ulong) -> előjel nélküli 64 bit
  • Single (float) -> 32 bites lebegőpontos (nem csak egész szám)
  • Double (double) -> 64 bites lebegőpontos (nem csak egész szám)
  • Decimal (decimal) -> 128 bites lebegőpontos (nem csak egész szám)

String adatok

  • Char (char)-> egy darab unicode karaktert tud tárolni (16 bit)
  • String (string)-> unicode karakterek halmaza

Tömbök

Előre megadott nagyságú adattároló struktúra. Homogén (csak egy típusú adatot tud tárolni).

Nagyságának megadása:

  • 1. int[] tomb tomb= new int[20] (20 db egész számnak van helye)
  • 2. int[] tomb = new int [20]

Már az elején megadott adatok:

  • int[] tomb= new int[5]{1,2,3,4,5}
  • int[] tomb={1,2,3,4,5} A tömb elemeinek indexe mindig 0-val kezdődik, tehát ha van öt elemünk, akkor az elemeink indexei 0-4 között lesznek. Ez fontos, hogy ne referáljunk olyan elemekre, amik nincsenek a tömbben (IndexOutOfRange)
