Vezérlési szerkezetek

Szelekciók

Szelekció (elágazás): olyan vezérlési szerkezet, amelyben egy logikai feltétel hatására a program végrehajtása két vagy több irányra bomlik szét. A feltételben szereplő kifejezésnek minden esetben logikai értéket kell visszaadnia.

Egy ágú szelekció

Feltételtől függő végrehajtás, azaz az egyirányú elágazás (ha akkor → if): a megadott feltétel teljesülésétől függ az adott szekvencia végrehajtása.

Kód:
string b = "nemüres"; if(Equals(b, "")) { b="BÁ"; } Console.WriteLine(b);
Kimenet:

nemüres

Kód:
int szam = 3; if(szam % 2 == 0) { Console.WriteLine("Ez a szám páros."); }
Kimenet:

(semmi, mert a szam nem felelt meg a feltételnek, így az "if"-en belüli rész nem futott le.)

Kód:
bool paros = true; int szam = 4 string b = "nemüres" if(paros) { Console.WriteLine(b + szam); }
Kimenet:

4nemüres

Két ágú szelekció

Döntéstől függő végrehajtás, azaz a kétirányú elágazás (ha akkor különben → if else): a megadott feltétel igazságtartalmától függően két különböző szekvencia végrehajtása között dönt a program.

Kód:
string b = "BÁ"; if(b == "") { b = "BÉ"; } else { Console.WriteLine(b); }
Kimenet:

BÁ

Kód:
int szam = 5; if(szam%2==0) { Console.WriteLine($"Ez a szám ({szam1}) páros. "); } else { Console.WriteLine($"Ez a szám ({szam1}) páratlan. "); }
Kimenet:

Ez a szám (5) páratlan.

Kód:
int szam = 120; string b = "BÁ"; if(szam > 100 && b.Length>8) { Console.WriteLine($"Hosszú szám: ({szam}), hosszú szó: ({b})."); } else { Console.WriteLine($"A Szám ({szam}) és/vagy a szó ({b}) kicsi (rövid)."); }
Kimenet:

A Szám (120) és/vagy a szó (BÁ) kicsi (rövid)."

Több ágú szelekció else if használatával

A kétirányú elágazás kibővíthető tetszés szerinti menyiséggel.

Kód:
int szam = Convert.ToInt32(Console.ReadLine()); if(szam < = 0) { Console.WriteLine("A szám nem lehet kevesebb 1-nél"); } else if(szam>= 5) { Console.WriteLine("A szám nem lehet nagyobb 5-nél"); } else { if(szam == 1) { Console.WriteLine("A szám egy"); } else if(szam == 2) { Console.WriteLine("A szám kettő"); } else if(szam == 3) { Console.WriteLine("a szám három"); } else if(szam == 4) { Console.WriteLine("a szám négy"); } else if(szam == 5) { Console.WriteLine("a szám öt"); }
Kimenet:

(a kimenet a felhasználó által megadott számtól függ.)

Kód:
string gyümi = "alma"; if(Equals(gyümi, "körte")) { Console.WriteLine("Ez egy körte, vigyázz nagyon finom"); } if(Equals(gyümi, "alma")) { Console.WriteLine("Ez egy alma, vigyázz nagyon finom"); } if(Equals(gyümi, "banán")) { Console.WriteLine("Ez egy banán, vigyázz nagyon finom"); } if(Equals(gyümi, "szőlő")) { Console.WriteLine("Ez egy szőlő, vigyázz nagyon finom"); }
Kimenet:

Kód:

Kimenet:

Több ágú szelekció switch case használatával

A kiválasztást többirányú elágazás, pontosabban érték szerinti elágazás (switch-case szerkezet) segítségével is megvalósíthatjuk, ez egyszerűbben olvasható kódot eredményez valamint picivel gyorsabban fut mint az if-elsek halmozása.

Kód:
string Gyümmi = "alma"; switch(Gyümmi) { case "körte": Console.WriteLine("A körte fincsi"); break; case "bálna": Console.WriteLine("A bÁLNA NAGY, és amúgy NEM GYÜMÖLCS"); break; case "paradicsom": Console.WriteLine("Ez egy paradicsom, gratulálok!"); break; case "alma": Console.WriteLine("Alma."); break; }
Kimenet:

Alma.

Kód:
int num = 1; switch(num) { case 156378: Console.WriteLine("Ez egy nagy szám"); break; case -156378: Console.WriteLine("Ez egy kis szám"); break; default: Console.WriteLine("A szám nem 156378 vagy -156378."); break; }
Kimenet:

A szám nem 156378 vagy -156378.

Kód:
int q = 8; switch (q) { case 2: Console.WriteLine("a 2 1. hatványa"); break; case 4: Console.WriteLine("a 2 2. hatványa"); break; case 8:; Console.WriteLine("a 2 3. hatványa"); break; default: Console.WriteLine("nem a 2-nek egy és három közötti hatványa"); break; }
Kimenet:

a 2 3. hatványa

Ciklusok

Iteráció (ciklus): olyan programszerkezet, amely egy utassítássorozatot ismétel.A ciklus részei: ciklusfej–a ciklusfejben határozzuk meg azokat a feltételeket, amelyek teljesülése esetén a program végrehajtja a ciklusmagot / ciklusmag–az a szekvencia, amelyet a program ismételten végrehajt.

Elöltesztelő ciklus(while)

A ciklusmag végrehajtását meghatározó feltétel (ez a bentmaradás feltétele) a ciklusfejben van, ennek vizsgálata egyszer mindenképpen megtörténik. Megjegyzés: Előfordulhat, hogy a ciklusmag sohasem hajtódik végre.Az elöltesztelő ciklus megvalósítása mindig előreolvasással történik, azaz a feltételben vizsgált adatot előbb meg kell határozni, és azután vizsgáljuk meg, hogya megadott érték megfelel-e a ciklusmag lefutási feltételének. A ciklusmagban pedig biztosítanunk kell a feltétel hamissá válását, különben végtelen ciklust kapunk.

Kód:
int bá = 0; while (bá < 4) { bá++; Console.WriteLine($"Bá={bá}"); }
Kimenet:

Bá = 1
Bá = 2
Bá = 3
Bá = 4

Kód:
int i=0; int c=0; while (i < 15) { c=i * i; i++; if (c % 3==0) { Console.WriteLine($"{c} az {i}-ből"); } }
Kimenet:

0 az 1-ből
9 az 4-ből
36 az 7-ből
81 az 10-ből
144 az 13-ből

Kód:
string j = ""; while (j!="aaaaa") { Console.WriteLine($"{j.Length}-edik index stringel számolva"); j += "a"; }
Kimenet:

0-edik index stringel számolva
1-edik index stringel számolva
2-edik index stringel számolva
3-edik index stringel számolva
4-edik index stringel számolva

Hátultesztelő ciklus

A hátultesztelőciklust általában ellenőrzött adatbevitelre alkalmazzuk, azaz addig folytatjuk egy meghatározott adat bekérését, ameddig az előírt határok közé nem esik, tehát a bekérés addig ismétlődik, amíg hibás a kapott érték.

Kód:
string[] magánhangzók = new string[] { "a", "á", "e", "é", "i", "í", "o", "ó", "ö", "ő", "u", "ú", "ü", "ű", "y" }; string megadott; do { Console.WriteLine("adjon meg egy magánhangzót"); megadott = Console.ReadLine(); } while (magánhangzók.Contains(megadott)); Console.WriteLine("ügyes vagy");
Kimenet:

Ha a felhasználó egy magánhangzót ad meg akkor a kimenet: "ügyes vagy", egyébként újrakéri

Kód:
int gyümiDb=324; do { Console.WriteLine("gyümölcsöt adtam hozzá"); gyümiDb++; }while (gyümiDb < 5); Console.WriteLine(gyümiDb);
Kimenet:

325

Kód:
int gyümiDB=324; do { Console.WriteLine("gyümölcsöt adtam hozzá"); gyümiDB++; } while (gyümiDB < 330); Console.WriteLine(gyümiDB);
Kimenet:

Bejáró ciklus(foreach)

Egy speciális ciklus, amelyet pl. tömbök és listák elemeinek bejárására találtak ki. A ciklus annyiszor fog lefutni, ahány elemű a lista, és a ciklus indexének értéke mindig megegyezik a bejárandó elem éppen soronkövetkező elemének értékével
